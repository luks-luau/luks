# luks-luau

A lightweight Luau runtime written in Rust with native module loading capabilities.

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)
![Rust](https://img.shields.io/badge/rust-1.70%2B-orange.svg)
![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20Linux%20%7C%20macOS%20%7C%20Android-blue.svg)

## Overview

luks-luau is a high-performance Luau runtime designed for general-purpose scripting. It allows you to execute Luau scripts and load native modules dynamically. It provides a C-compatible API for easy integration with other languages and includes a command-line interface for direct script execution.
